Bit By Query is a hands-on DBMS competition designed for students who enjoy working directly with data and SQL. What started in 2024 as an idea to host database-focused competitions has now evolved into a fully functional competitive platform, built end-to-end by the AISSMS IOIT Web Team, exclusively to support this event.

In Bit By Query, participants solve real-world DBMS problems by writing and executing SQL queries on a live dashboard. Each problem is backed by a structured database, and solutions are evaluated automatically based on correctness, syntax, and output precision.
Competition Format
  • Write and execute SQL queries directly on the platform
  • Each query runs against a predefined problem database
  • Solutions are auto-judged by matching exact output
  • For INSERT / UPDATE / DELETE problems, submit two queries separated by a semicolon
  • For SELECT problems, submit a single query
  • MySQL syntax is used throughout the competition
The platform also features real-time problem states, a countdown-based contest flow, and instant feedback, making the experience both competitive and practical.
The entire competition platform is open source. You can explore how it’s built and how the judging works here:
